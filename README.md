# Bank Customer Churn Analysis Dashboard

## Project Objective
This project analyzes bank customer data to understand churn patterns and build an interactive dashboard using Excel, SQL, and Power BI. The goal is to identify key factors affecting customer churn and provide actionable business insights.

---

## Dataset Information
- Total Records: 10,000
- Target Column: Exited (0 = Retained, 1 = Churned)
- Key Features:
  CreditScore, Geography, Gender, Age, Balance, Tenure, NumOfProducts, HasCrCard, IsActiveMember, EstimatedSalary

---

## Tools Used
- Microsoft Excel
- SQL (PostgreSQL)
- Power BI

---

## Project Workflow
- Data Import and Cleaning
- Data Validation using SQL
- Feature Engineering (Age Group, Balance Category, Credit Category)
- KPI Calculations in Excel
- Pivot Tables & Summary Tables
- Interactive Dashboard in Excel & Power BI
- Business Insights Generation

---

## SQL Analysis Summary
- Churn analysis by Geography, Gender, Age Group
- Credit Score and Balance analysis
- Active vs Inactive customer analysis
- Customer segmentation using CASE statements
- Window functions for ranking and comparison
- CTE-based analysis for aggregated insights

---

## Excel Dashboard Features
- KPI Cards (Total Customers, Churn Rate, Active/Inactive Members)
- Category-wise churn analysis
- Interactive charts and visuals
- Business performance tracking dashboard

---

## Power BI Dashboard Features
- KPI Overview Cards
- Column Charts, Bar Charts and Pie Charts
- Geographic and demographic analysis
- Interactive slicers (Gender, Age Group, Geography, Credit Category)
- Business Insights section

---

## Key Insights
- Germany has the highest churn rate
- Senior customers are most likely to churn
- Inactive members churn significantly more than active members
- Low balance customers show higher churn rate
- Poor credit score customers are at higher risk
- Credit card ownership has minimal impact on churn

---

## Final Conclusion
This project demonstrates end-to-end data analysis skills using SQL, Excel, and Power BI. The analysis successfully identifies key factors affecting customer churn and helps in building data-driven business decisions for customer retention.
